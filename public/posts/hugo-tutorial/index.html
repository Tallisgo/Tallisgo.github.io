Hugo博客搭建教程

详细介绍如何使用Hugo搭建个人博客的完整教程。
January 2, 2024 · Tallis

Hugo博客搭建完整教程#

本教程将带你从零开始搭建一个基于Hugo的个人博客。

1. 安装Hugo#

macOS#

1

brew install hugo

Windows#

1

choco install hugo

Linux#

1

sudo apt-get install hugo

2. 创建新站点#

1
2

hugo new site my-blog
cd my-blog

3. 添加主题#

1

git submodule add --depth=1 https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod

4. 配置hugo.toml#

在hugo.toml文件中添加主题配置：

1
2
3
4
5
6

theme = 'PaperMod'

[params]
  title = '我的博客'
  description = '个人博客描述'
  author = '你的名字'

5. 创建内容#

1

hugo new posts/我的第一篇文章.md

6. 本地预览#

1

hugo server -D

访问 http://localhost:1313 查看你的博客。

7. 构建静态文件#

1

hugo

构建完成后，静态文件将生成在public目录中。

总结#

通过以上步骤，你就可以成功搭建一个Hugo博客了。Hugo的强大之处在于它的速度和灵活性，非常适合个人博客和文档网站。
